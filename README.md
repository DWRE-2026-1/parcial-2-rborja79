# Examen Parcial: Calculadora de Teclado Numérico con React.js

## Descripción del Problema
El objetivo de este examen es evaluar tus habilidades en el desarrollo de aplicaciones web utilizando **React.js**. Deberás construir una interfaz funcional que presente un **teclado numérico** en pantalla, permitiendo al usuario realizar operaciones aritméticas básicas entre dos números.

La aplicación debe ser capaz de procesar las siguientes operaciones:
* Suma (+)
* Resta (-)
* Multiplicación (*)
* División (/)

## Requisitos Técnicos
1.  **Componentización:** El teclado y la pantalla de visualización deben estar organizados de forma lógica mediante componentes de React.
2.  **Manejo de Estado:** Utilizar hooks (`useState`) para gestionar la entrada de números y el resultado de las operaciones.
3.  **Interactividad:** No se permite el uso del teclado físico para ingresar números; toda la interacción debe ocurrir a través de los clics en el teclado numérico de la interfaz.
4.  **Validaciones:** El sistema debe manejar casos especiales, como la división por cero.

## Reglas del Examen
* **Modalidad:** Individual.
* **Uso de IA:** Se permite (y se incentiva) el uso de herramientas de Inteligencia Artificial (ChatGPT, GitHub Copilot, Claude, etc.). Sin embargo, es obligatorio documentar el proceso.
* **Entrega:** El proyecto debe estar desplegado en una plataforma (Vercel, Netlify o GitHub Pages) y el código fuente debe estar en un repositorio de GitHub.

## Registro de Prompts (Bitácora)
Al final de este archivo o en un archivo adjunto llamado `PROMPTS.md`, deberás incluir el listado de los prompts principales que utilizaste para generar código, resolver errores o diseñar la interfaz.
